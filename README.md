# YouTube影片下載器
## 聲明
- 本軟體使用`Pytube`來下載YouTube上的影片
- 版本號命名規則使用[Semantic Versioning](https://semver.org/)
## 功能
- 將YouTube上的影片轉為mp3檔案,mp4檔案
- 無限制
## 如何使用
- 1.下載至您的裝置([這裡下載]())
- 2.解壓縮
- 3.開啟資料夾，並點開`YouTube Downloads Tool.exe`
- 4.將您在YouTube上找到的`影片連結`複製到`連結輸入框`
- 5.選擇您要儲存的`路徑`
- 6.點擊`轉換`按鈕
- 7.選擇`mp3`或`mp4`檔案
- 8.**等待您的檔案!!**
